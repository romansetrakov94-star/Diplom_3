# Diplom_3 — Автоматизация тестирования веб-приложения

Автотесты для веб-приложения на Python с использованием PyTest и Selenium WebDriver.

---

## 📌 О проекте

Данный проект выполнен в рамках дипломной работы по курсу «Инженер по тестированию» в Яндекс.Практикуме.  
Цель — автоматизировать проверку ключевого функционала веб-приложения (интернет-магазин).

**Что было сделано:**
- Реализованы сценарии: авторизация, поиск товаров, добавление в корзину, оформление заказа.
- Использован паттерн Page Object Model для удобства поддержки.
- Настроены отчёты Allure для визуализации результатов.

---

## 🛠️ Стек технологий

- **Python** 3.10
- **PyTest** — фреймворк для тестирования
- **Selenium WebDriver** — управление браузером
- **Page Object Model** — структура тестов
- **Allure** — отчёты
- **Git** — контроль версий

---

## 🚀 Запуск тестов

```bash
# Клонировать репозиторий
git clone https://github.com/romansetrakov94-star/Diplom_3.git

# Перейти в папку проекта
cd Diplom_3

# Установить зависимости
pip install -r requirements.txt

# Запустить все тесты
pytest -v

# Сгенерировать отчёт Allure (если установлен Allure)
allure serve allure-results/
